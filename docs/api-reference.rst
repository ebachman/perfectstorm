Perfect Storm API Reference
===========================

Groups (`/v1/groups`)
---------------------

Create
~~~~~~

**POST /v1/groups**

List
~~~~

**GET /v1/groups**

Read
~~~~

**GET /v1/groups/$name**

Update
~~~~~~

**PUT /v1/groups/$name** or **PATCH /v1/groups/$name**

Delete
~~~~~~

**DELETE /v1/groups/$name**
